1️⃣以後新增按鈕的兩種方式

方式1：手動改原始程式碼（推薦固定外部連結用）
1.HTML 按鈕區新增一行
在 <div class="tab-bar" id="tabBar"> 裡面加：

<button class="tab-btn " type="button" data-src="你的完整網址" data-name="按鈕顯示文字">按鈕顯示文字</button>

2️⃣JS 的 APP_CONFIG.tools 同步新增
找到這段：

const APP_CONFIG = {
  ...
  tools: [
    { name: 'TLK', src: 'https://tlk.io/pcb' },
    { name: '飛書', src: 'https://tcn7suhgzdxm.feishu.cn/drive/home/' },
    { name: '測試', src: 'https://fam3000.github.io/op/good/網域.html' },
    // 在這一行下方新增
    { name: '按鈕文字', src: '完整https網址' }
  ]
};

方式2：使用頁面設定彈窗（僅適用本地 .html 檔案）

點擊左上角「常用」文字打開設定視窗，在「批量按鈕清單」文字框每行輸入一個名稱，系統會自動生成同檔名的本地 html 按鈕；

注意：彈窗批量新增只會自動生成本地檔案（xxx.html），無法自動填入外部 https 網址，外部連結只能用方式1手動寫死。

3️⃣、運作規則說明

1. data-src 開頭為 http:// 或 https:// → 點擊自動新分頁開啟網頁，不會載入下方 iframe；

2. data-src 是 xxx.html 這類本地檔名 → 載入到下方 iframe 區域；

3. 所有按鈕樣式、點擊縮放、滾動條都會沿用現有樣式。
